LogAnalyzer v1.0
================

Terminal-based log analysis tool with timezone conversion and keyword search.

Usage:
  loganalyzer                    # Interactive mode
  loganalyzer -f <file>         # Analyze file
  loganalyzer -k <keyword>      # Search keyword
  loganalyzer -t IST|UTC        # Set timezone

For more information, run: loganalyzer --help
